# Catalan Flag Snake Game - Winter Edition

A visually immersive snake game built with Pygame.

## Features
- Catalan flag snake (red/yellow)
- Animated winter background (birds & clouds)
- Pause (spacebar) and restart (R key)
- Crash flash animation
- Dramatic game over with Excalidrax font
- Sound effect placeholder

## How to Run
1. Install requirements: `pip install pygame`
2. Place Excalidrax.ttf font in `assets/` folder.
3. Run: `python main.py` from `utils/snake_game/`

## Controls
- Arrow keys: Move snake
- Spacebar: Pause
- R: Restart after game over

## Folder Structure
- `main.py` - Entry point
- `game.py` - Game logic
- `snake.py` - Snake logic
- `food.py` - Food logic
- `background.py` - Animated background
- `config.py` - Settings
- `assets/` - Font and sound files
